## Clone
```bash
git clone https://github.com/alexalbuquerque87/pythonando_django.git
```

## Primeiros passos

# Criando venv
```bash
python -m venv venv
```
# Ativando venv powershell
```bash
venv\Scripts\activate
```
# Instalando Django
```bash
pip install django
```
# Start project
```bash
django-admin startproject aula_django .
```
