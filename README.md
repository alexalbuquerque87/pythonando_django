## Clone
```bash
git clone https://github.com/alexalbuquerque87/pythonando_django.git
```

## Primeiros passos

```bash
# Criando venv
python -m venv venv

# Ativando venv powershell
venv\Scripts\activate

# Instalando Django
pip install django

# Start project
django-admin startproject aula_django .
```
